# WASSUP
WASSUP est une application web gratuite qui permet à chaque utilisateur de découvrir en temps réel tous les événements disponibles autour de lui selon sa position géographique.
Sorties en boîte, concerts, spectacles, soirées privées, expositions, festivals et bien plus encore… tout est réuni en un seul endroit.

Les utilisateurs peuvent se connecter pour :

voir instantanément tous les événements du jour autour d’eux,

ajuster le rayon de recherche (10 à 50 km),

accéder aux informations détaillées de chaque événement,

contacter le promoteur et obtenir l’itinéraire via Google Maps.

Une interface dédiée permet aux promoteurs d’événements de :

créer un compte,

publier leurs événements avec lieu, date, description et tarifs,

gérer leurs pass,

recevoir des messages et échanger avec les participants.

WASSUP simplifie la vie des sortants et donne de la visibilité aux organisateurs, le tout gratuitement et avec une interface moderne et intuitive.
